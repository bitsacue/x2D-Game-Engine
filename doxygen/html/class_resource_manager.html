Detailed Description

This class handles resource loading and handling.

Resources in this context are objects which are loaded from the hard drive and which should only be read once.

Resources are loaded as follows

ResouceClassPtr resource = ResourceManager::get<ResourceClass>(":/path_to_file.res");

Where ResourceClass is the name the class of the type of resource you are trying to load. ResouceManager can currently load image files and font files.
