\subsection{Detailed Description}
The \hyperlink{class_texture}{Texture} class provides a interface to load textures into memory and drawing them on-\/screen. 

The texture class stores the Open\-G\-L texture I\-D of a loaded texture. You can load a texture by constructing an instance of this class, or by calling G\-F\-X.\-load\-Texture()


\begin{DoxyCode}
CTexture @texture = @CTexture(\textcolor{stringliteral}{"images/a\_texture.png"});
\end{DoxyCode}


\begin{DoxyRefDesc}{Todo}
\item[\hyperlink{todo__todo000001}{Todo}]We can currently only load png textures, but more format support is planed for future versions.\end{DoxyRefDesc}
