\subsection{Detailed Description}
This class handles resource loading and handling. 

Resources in this context are objects which are loaded from the hard drive and which should only be read once.

Resources are loaded as follows
\begin{DoxyCode}
ResouceClassPtr resource = ResourceManager::get<ResourceClass>(\textcolor{stringliteral}{":/path\_to\_file.res"}); 
\end{DoxyCode}
 Where Resource\+Class is the name the class of the type of resource you are trying to load. Resouce\+Manager can currently load image files and font files.
